apageapy

apageapy is a web application framework designed for building web applications and restful apis using the Python programming language. apageapy is powered by Starlette, an amazing ASGI framework that is lightweight, fast, and flexible that powers other excellent frameworks like FastAPI and Django Rest Framework.

What makes apageapy unique is that it is designed around the DRY (Don't Repeat Yourself) principle. and is more likely to appeal to developers who prefer configuration over convention, but without the complexity. It's ultimately designed for anyone who wants to build web applications and/or restful apis with minimal boilerplate code, while maximizing the reusability of their code.
